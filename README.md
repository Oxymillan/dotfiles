# My dotfiles
Very minimal config
Color scheme: Nord

Some dependency config like picom or polybar will only work on Arch.
### Dependencies:
- zsh
- picom
- polybar
- awesomeWM
  - Collision: https://github.com/Elv13/collision
- rofi
- NeoVim
- Nerd Font
- nitrogen
- colorls

### Usage:
Custom Rofi theme:
Copy the nord.conf file to /usr/share/rofi/themes
